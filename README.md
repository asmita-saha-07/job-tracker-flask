# Job Application Tracker

A simple Flask web application to track job applications, including company details, roles, notes, and application status.

---

## Features

- Add new job applications
- View all saved applications
- Update job details
- Delete applications
- Track application status (Accepted, Rejected, Interview, Offered)

---

## Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- Bootstrap 5
- HTML (Jinja2 templates)
